## Get Sky-on

The easiest, and fastest way to get the latest beta releases is to use our [github releases](https://github.com/Sky-on/Sky-on-Android/releases). It is recommend to use these with [Obtainum](https://github.com/ImranR98/Obtainium) to get the latest updates.

Alternatively, these other providers are also available, but may be slower to update.

If you encounter any problems or have questions, [ask us on the discord](https://discord.gg/Sky-on), [create an issue](https://github.com/Sky-on/Sky-on-Android/issues), or [post in the forum](https://github.com/orgs/Sky-on/discussions) and we'll help as we can.

## Documentation

https://Sky-on.org/docs/development/android/

Note: when building the `google` flavor locally you will need to supply your own [Google Maps Android SDK api key](https://developers.google.com/maps/documentation/android-sdk/get-api-key) `MAPS_API_KEY` in `local.properties` in order to use Google Maps.
e.g.
```properties
MAPS_API_KEY=your_google_maps_api_key_here
```
